📖 User guide

1. Add your API keys (optional)

Open Settings and paste your Anthropic API key to enable the AI analysis, and a Google PageSpeed key for higher Core Web Vitals quota. Keys live only in your browser.

2. Run an audit

  1. Enter the client's website URL (and optionally a client name, target keyword, page limit).
  2. Toggle which checks to run: link checking, Core Web Vitals, and AI analysis.
  3. Click Run audit and watch the live progress: crawl → links → vitals → analysis.

3. Read the report

  • Overall & category scores (0–100) across all 7 categories.
  • Core Web Vitals (LCP, CLS, INP, TBT) from real Google data.
  • Quick wins and a prioritised P1/P2/P3 action plan with a fix for each issue.

4. Export

Download the branded report as HTML, PDF, or Word (.docx) from the results header.

No key? You still get the full deterministic rules-engine audit — it just won't include the AI's extra narrative.

About

The Website Audit & Recommendation System scans a client's website and produces a branded, prioritised report across 7 categories: Technical SEO, On-Page SEO, Performance, Content, Local SEO, UX & Conversion, and Accessibility.

Backend: Python · FastAPI · httpx · BeautifulSoup. Performance from Google PageSpeed Insights. AI analysis via Claude (structured outputs). Reports as PDF, Word and HTML. This is the lightweight single-file HTML frontend.
